# BasePlay UI Style Guide

## Cores
- Primary: `#7A1126`
- Primary Hover: `#5C0D1D`
- Soft: `#F4D6DB`
- Background: `#FFFFFF`
- Surface: `#F6F7F9`
- Text: `#1F2933`
- Muted: `#6B7280`
- Border: `#E6E1D9`

## Tipografia
- Fonte base: `Inter`, `Poppins`, `Segoe UI`, `sans-serif`

## Botoes
- Primario: `.bp-btn-primary`
- Secundario: `.bp-btn-secondary`
- Perigo: `.bp-btn-danger`
- Ghost: `.bp-btn-ghost`

## Badges
- Base: `.bp-badge`
- Variacoes: `.bp-badge-success`, `.bp-badge-warn`, `.bp-badge-danger`, `.bp-badge-info`

## Inputs
- `.bp-input`, `.bp-select`, `.bp-textarea`

## Cards
- `.bp-card`, `.bp-card-header`, `.bp-card-body`

## Toast
- Use `window.bpToast('success', 'Mensagem')`
- Tipos: `success`, `error`
