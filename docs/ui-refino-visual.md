# Refinamento Visual BasePlay

## Cores
- Primária (vinho): #7A1126
- Hover: #5C0D1D
- Soft: #F4D6DB
- Fundo: #FFFFFF
- Fundo suave: #F6F7F9
- Texto: #1F2933

## Fonte
- Poppins (Google Fonts)

## Componentes
- Botões: primário vinho, secundário neutro, perigo vinho escuro.
- Cards: borda suave, sombra leve, cantos arredondados.
- Tabelas: cabeçalho claro, linhas zebra, hover sutil.
- Formulários: inputs maiores, foco em vinho.
- Badges: cores suaves para status.

## Layout
- Sidebar branca com item ativo vinho.
- Topbar limpa com usuário.
- Footer discreto.
- Dashboard com cards em grid e ícones sutis.

## Boas práticas
- Manter contraste alto em telas de campo.
- Evitar excesso de cores e sombras.
- Priorizar leitura rápida.

## Arquivo de tema
- `public/assets/css/baseplay-theme.css`
